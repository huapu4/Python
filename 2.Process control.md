## 分支语句  
> ### 条件判断  
>> if  
>> 命令逻辑：如果……就……  
>> 命令格式：  
```
if XXX:
    # 确定条件
    print('123')
        # 输出结果
```
>> 命令格式注意事项：  
>>> if确定条件后结尾加英文“:”  
>>> 回车后空4格+要执行的代码（eg：print）  
> ### 双向判断  
>> if...else...  
>> 命令逻辑  
>>> 如果不满足if的条件，就执行else的命令  
>>> else要有一个平级的前提条件（if）  
>>> ![image](https://github.com/huapu4/Python/blob/master/images/2.png)   
>> 命令格式  
```
if XXX:
    # 确定条件
    print('123')
        # 输出结果1
else:
    # 不满足if条件时
    print('321')
        # 输出结果2
```
>> 命令格式注意事项  
>>>  if、else结尾+英文“:”  
>>>  if、else回车后，空4格+要执行的代码（eg：print）  
>>>  else要与if对齐  
> ### 多向判断  
>> if...elif...else...  
>> 命令逻辑  
>>> 如果不满足if的条件，就判断是否满足elif下的命令，如果满足，就是执行；如果不满足，就执行else下的命令  
>>> elif不一定有else，但是一定要有if  
>>> ![image](https://github.com/huapu4/Python/blob/master/images/3.png)   
>> 命令格式
```
if XXX:
    # 确定条件1
    print('123')
        # 输出结果1
elif YYY:
    # 确定条件2
    print('213')
        # 输出结果2
else:
    # 不满足if条件时
    print('321')
        # 输出结果3
```
>> 命令格式注意事项  
>>> if、elif、else结尾+英文“:”  
>>> if、elif、else回车后，空4格+要执行的代码  
>>> if、elif、else三者对齐  
> ###  if循环嵌套  
>> 条件里面套条件，在已有条件下附加额外的条件  
>> 命令逻辑  
>>> 根据缩进判断层次  
>>> 根据层级执行命令  
>>> 根据赋值判断是否符合条件，符合执行条件下命令，不符合进入下一判断  
>>> ![image](https://github.com/huapu4/Python/blob/master/images/4.png)  
>> 命令格式  
```
if XXX：:
    # 确定条件1
    if XXXX:
        # 附加条件1
        print('123')
            # 输出结果1
    else:
        # 不满足附加条件1
        print('456')
            # 输出结果2
else:
    # 不符合确定条件1
    if YYY:
        # 满足附加条件2
        if YYYY:
            # 不满足确定条件1，满足附加条件2、3
            print('321')
                # 输出结果3
        else:
            # 不满足附加条件3
            print('654')
                # 输出结果4
    else:
        # 不满足确定条件1与附加条件2
        print('123456')
            # 输出结果5
```
>> 命令格式注意事项  
>>> 额外条件需要缩进，每次缩进都是4个空格  
>>> 同等级if、else条件对整齐  
## 循环语句  
> ### while循环  
>> whlie...  
>>> 在给定的判断条件为 true ，不断执行循环，直到指定的条件不满足时，退出循环体。  
>>> ![image](https://github.com/huapu4/Python/blob/master/images/5.png)  
>>> 命令格式  
'''
while XXX:
    # 满足循环条件
    print('123')
    # 输出结果  
'''
>> whlie...死循环  
>>> 如果条件判断语句永远为 true，循环将会无限的执行下去 
>> whlie...break  
>>> 终止本层循环，提前结束整个循环  
>>> ![image](https://github.com/huapu4/Python/blob/master/images/6.png)  
>>> 命令格式  
```
while XXX:
    # 满足循环条件
    if XXXX:
        # 附加条件
        print('123')
        break
            # 输出结果，终止循环
print('end')
    # 循环终止
```
>> while...continue...  
>>> 终止本次循环，继续下一次循环  
>>> ![image](https://github.com/huapu4/Python/blob/master/images/7.png)  
>>> 命令格式  
```
while XXX:
    # 满足循环条件
    if XXXX:
        # 附加条件
        print('123')
        continue
            # 输出结果，结束本次循环
print('end')
    # 循环终止
```
>> while...else...  
>>> 正常结束循环后，执行else后的结果  
>>> ![image](https://github.com/huapu4/Python/blob/master/images/8.png)  
>>> 命令格式 
```
while XXX:
    XXXXXXX
    print('123')
        # 执行循环并输出结果
else:
    # 正常结束循环
    print('456')
        # 输出结果
pritn('123456')
    # 输出结果
```
>> 命令格式注意事项  
>>> 循环体内的代码，缩进4个空格  
>>> 同等级的循环，代码对齐  
>>> while、else结束后必须跟英文冒号——“:”  
>>> 循环体内部（break、ccontinue）后不需要跟冒号  
> ### for 循环  
>> for...in...  
>>> 遍历任何序列的内容  
>>> ![image](https://github.com/huapu4/Python/blob/master/images/9.png)  
>>> 命令格式  
```
for i in j:
    # j: 列表、字符串、元祖、字典、集合等序列类型
    XXXX

 print('123')
```

>> for...break...  
>>> 终止本层遍历，提前结束整个遍历  
>> for...continue...  
>>> 终止本次遍历，继续下一次遍历  
>> for...else...

>> 知道循环次数的时候，优先用for循环，不知道循环次数的时候用while  
## 流程控制语句用法说明  
>> 避免多层分支嵌套  
>>> 竭尽所能的避免分支嵌套  
>>> 多层嵌套代码可读性差  
>> 封装过于复杂的逻辑判断  
>>> 出现太多 not/and/or，可读性差→用not 的形式取反  
>> 避免重复代码  
>>> 注意分支代码中的重复代码块  
>>> 重复代码降低代码质量  
>> “德摩根定律”
>>> not A or not B =not (A and B)  
>> 在条件判断中使用 all() / any()  
>>> all(sequence→seq)：仅当 seq 中所有对象都为布尔真时返回 True，否则返回 False  
>>> any(sequence→seq)：只要 seq 中任何一个对象为布尔真就返回 True，否则返回 False  
>> 使用 try/while/for 中 else 分支
>>> 所有 try/while/for语句正常执行后，才执行 else 分支下的代码。  
>> 与 None 值的比较  
>>> None 在 Python 语言中是一个单例对象  
>>> 比较变量的方法  
>>>> ==：表示二者所指向的的值是否一致  
>>>> is：表示二者是否指向内存中的同一份内容，也就是 id(x) 是否等于 id(y)  
>>> 当要判断某个变量是否为 None 时，使用 is  
>> 留意 and 和 or 的运算优先级  
>>> and 运算符的优先级大于 or
